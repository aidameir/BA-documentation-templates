# BRD — E-commerce Checkout Redesign

## 1. Executive Summary
The checkout process currently has a 35% abandonment rate. The goal is to redesign the checkout to increase completed orders.

## 2. Business Objectives
- Reduce checkout abandonment from 35% → 20%
- Reduce checkout time by 40%
- Improve mobile UX

## 4. Scope
### In Scope
- Checkout UI redesign
- Guest checkout
- 1-click payment

### Out of Scope
- Loyalty program
- New shipping carriers

## 5. As-Is Process
User must create an account → multiple steps → slow load times.

## 6. To-Be Process
- Guest mode
- Payment selection on first screen
- Address auto-fill

## 7. Requirements
BR-001: User must be able to check out as a guest  
BR-002: System must support Apple Pay/Google Pay  
BR-003: Auto-fill customer shipping address

## 8. KPIs
- +20% completed checkouts
- –40% page load time

## 9. Risks
- Payment gateway delays
